# Ekran görüntüleri

README ve GitHub sayfasında kullanılan görseller bu klasördedir.

## Yeniden oluşturma

```bash
# Uygulama otomatik başlar; .env.local içinde E2E kullanıcısı dashboard görselleri için gerekli
npm run screenshots
npm run screenshots:gif   # opsiyonel — ffmpeg ile demo.gif
```

## Dosyalar

| Dosya | Açıklama |
|-------|----------|
| `demo.gif` | Landing turu animasyonu |
| `landing-hero.png` | Ana sayfa üst bölüm |
| `landing-how-it-works.png` | Nasıl çalışır |
| `landing-pricing.png` | Fiyatlandırma |
| `dashboard.png` | Kullanıcı paneli |
| `templates.png` | Şablon listesi |
| `upload.png` | Belge yükleme |

`gif-frames/` geçici kareler içerir; Git'e eklenmez.
